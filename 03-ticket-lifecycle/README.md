# Phase 3: Ticket Lifecycle Scenarios

The core of this project — 10 end-to-end ticket scenarios demonstrating real-world help desk workflows.

## How It Works

For each scenario, both roles are played:
1. **End user** submits a ticket from the Windows 11 client
2. **Agent** triages, works, and resolves the ticket in osTicket
3. **Technical work** is performed on Windows Server or Windows 11 as needed
